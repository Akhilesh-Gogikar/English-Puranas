# Puranas Redesigned by Demographic

This folder contains **1313** redesigned story-analysis files generated from `puranas_jsons/`.

## Structure
- One file per original Purana JSON (same filename).
- Each file includes:
  - `analysis` (core summary, themes, audience notes, sensitivity notes)
  - `redesigned_story_by_demographic` with versions for:
    - children (8-12)
    - teens (13-17)
    - young adults (18-30)
    - devotional/traditional readers
    - global general audience

## Goal
Preserve the core narrative while making each story easier to understand across different reader groups.
